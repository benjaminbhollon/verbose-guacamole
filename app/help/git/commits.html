What are Commits?

A guide for writers

Verbose Guacamole uses Git, a version control software originally designed for keeping track of changes to computer code. As such, Git has powerful performance and allows integration with a multitude of tools.

The most important feature of Git, from the standpoint of a writer using Verbose Guacamole, is the commit functionality. A commit is a set of changes that you save that you can refer back to it later. Think of it as a saved copy of an older version.

To create a commit, type a short description of what you changed (a commit message) in the box labelled "Commit message" and click "Commit".

It is typically a good idea to try to write commit messages that are unique so you can tell them apart.

A feature already possible for people who know how to use Git is to set up a remote repository to sync your changes between devices. A guide for this as well as built in features will come soon.
